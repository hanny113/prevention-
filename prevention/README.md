# 바이러스 방폭 게임: 인류의 마지막 희망

신종 감염병으로부터 인류를 구하는 방폭탄해제 게임입니다.

## 게임 소개

전 세계는 정체불명의 신종 감염병으로 인해 위기에 빠졌습니다. 악당이 감염병을 막을 단서를 담은 백신 알고리즘을 지뢰처럼 숨겨 놓았습니다. 당신은 올바른 감염병 지식을 활용해 단계별 코드를 해제하며 인류를 구해야 합니다.

## 게임 방법

1. '게임 시작' 버튼을 클릭하여 게임을 시작합니다.
2. 각 레벨마다 감염병 관련 퍼즐이 제시됩니다.
3. 퍼즐을 풀어 정답을 입력창에 입력하고 '제출' 버튼을 클릭합니다.
4. 정답이면 다음 레벨로 진행하고, 오답이면 감염률이 증가하고 시간이 줄어듭니다.
5. 힌트가 필요하면 '힌트 보기' 버튼을 클릭할 수 있지만, 시간이 차감됩니다.
6. 5개의 레벨을 모두 클리어하면 게임 성공, 시간이 다 되거나 감염률이 100%에 도달하면 게임 오버입니다.

## 게임 특징

- 실제 감염병 관련 지식을 활용한 교육적 퍼즐
- 타이머와 감염률 시스템으로 긴장감 조성
- 오답 입력 시 바이러스가 퍼지는 애니메이션 효과
- 레벨별로 난이도가 증가하는 구조
- 모바일 기기에서도 플레이 가능한 반응형 디자인

## 실행 방법

1. 이 저장소의 파일들을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 열어 게임을 시작합니다.

## 게임 화면

- 시작 화면: 게임 배경 스토리와 시작 버튼을 제공합니다.
- 게임 화면: 퍼즐, 타이머, 감염률 표시기, 정답 입력창이 있습니다.
- 결과 화면: 성공 또는 실패 메시지와 재시작 버튼을 제공합니다.

## 개발 환경

- JavaScript (ES6+)
